The Daily X

Link to Github Repo
HTML, CSS, Bootstrap, jQuery

The Daily X can be used to track daily activities that you want to encourage or discourage. Currently this app only works for a single day. Users can click in boxes to designate that category was done. Each box represents a single unit or 100 calories. Soon, these details will be configurable. The reset button clears all 'X's from the page and allows you to begin a new day fresh. Future features will include badges for success and streaks of successful days for each category.
