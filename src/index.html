The Daily X

  • Bootstrap
  • JQuery
  • JavaScript
  • Git
Link to Github Repo

The Daily X can be used to track daily activities that you want to encourage or discourage. Currently this app only works for a single day. Users can click in boxes to designate that category was done. Each box represents a single unit or 100 calories. Soon, these details will be configurable. The reset button clears all 'X's from the page and allows you to begin a new day fresh. Future features will include badges for success and streaks of successful days for each category.

Cofounders is the boardgame for startup founders and entrepreneurs. CollabaLearn is a framework that connects anyone who is learning how to code with others who want to collaborate on a learning project. With the ability to find mentors and to pair program using projects in GitHub, users can stay home and collaborate while beefing up skills on real projects. The framework integrates with meetup groups, code schools, online tutorial frameworks and more.

Link to Public Website (code available by request only)

Mixteka is a virtual DJ Solution for weddings and events. This product was built to be a do-it-yourself solution that combines the expertise of a DJ with the pricepoint of DIY. An upgrade to using personal players and online music providers, Mixteka allowed users to invite guests to recommend music, and for the organizer to pick and choose based on the DJ curated options, a 5M song library and the recommendations of friends. The mobile player ensured your music did not stop and that your device did not interrupt playback. Mixteka has ceased operations due to lack of music licensing. Look for a free version if the team can take the time to connect to the Spotify API in their extra time.
